这是一个简单的网上书城的功能小项目。主要是用javaweb基础最原始的技术实现的。

包含中的内容有：

html/css/javascript/xml

/servlet/jsp/mysql/jdbc/mvc等内容。

这是自己学习中慢慢总结的代码，一个代码积累。学习javaweb基本知识汇总。

包含前台和后台的功能。实现的也是比较简单，注册和登录，当然没有做MD5加密之类的。

就是单纯的javaweb基础知识功能。
